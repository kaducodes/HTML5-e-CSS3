como esse, você pode usar a tag <br>.

formatos de imagens para web: .jpeg e .png (poucas vezes .gif)
.jpeg : criado em 1983. vantagem: consegue compactar uma imagem.
.png : criado em 1996. vantagem: usado para substituir o .gif permitindo transparência (sem animação). menos compactado do que o .jpeg
